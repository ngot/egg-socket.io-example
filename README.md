A simple example of egg-socket.io

## Install

```bash
$ npm install
```

## Requirements

- Node.js >= 8.0

## Start

### Dev mode

```bash
$ npm run dev
```

### Production mode

```bash
$ npm start
```

## Test

```bash
$ npm test
```
